===================================================
ASSISTENTE DE VOZ CHICA COM AVATAR - MANUAL
===================================================

📋 REQUISITOS DO SISTEMA
===================================================

1. 🐍 Python 3.8 ou superior
2. 💾 8GB RAM mínimo (16GB recomendado)
3. 🎤 Microfone funcionando
4. 🔊 Alto-falantes funcionando
5. 🌐 Conexão com internet (para baixar modelos)
6. 🖥️ Sistema gráfico (para o avatar) — opcional

⚙️ INSTALAÇÃO DAS DEPENDÊNCIAS
===================================================

1. Instale o Python 3.8+ se ainda não tiver:
   - Windows: https://www.python.org/downloads/
   - macOS: `brew install python`
   - Linux: `sudo apt install python3 python3-pip`

2. Crie um ambiente virtual (recomendado):
   ```bash
   python -m venv venv

   # Windows
   venv\Scripts\activate

   # macOS/Linux
   source venv/bin/activate
   ```

3. Instale as dependências Python:
   ```bash
   pip install -r requirements.txt
   ```

4. Instale dependências do sistema para o Pygame (avatar):
   **macOS:**
   ```bash
   brew install sdl2 sdl2_image sdl2_mixer sdl2_ttf portaudio
   ```
   **Ubuntu/Debian:**
   ```bash
   sudo apt-get install python3-dev libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev portaudio19-dev
   ```

🔧 PROVEDOR LLM (ESCOLHA UM)
===================================================

O Chica suporta dois provedores de LLM local.

Opção A — Ollama (padrão)
--------------------------
1. Baixe e instale: https://ollama.ai/
2. Inicie o serviço: `ollama serve`
3. Baixe um modelo: `ollama pull qwen3:1.7b`

Opção B — LM Studio (alternativa)
----------------------------------
1. Baixe e instale: https://lmstudio.ai/
2. Carregue um modelo na interface
3. Habilite a API: Settings > Enable API (porta 1234)
4. Instale a dependência: `pip install openai`
5. Configure no config.py:
   ```python
   LLM_PROVIDER = 'lm_studio'
   LLM_MODEL = 'bonsai-8b'  # ou o modelo que carregou
   ```

🎯 CONFIGURAÇÃO
===================================================

Todas as configurações estão centralizadas no arquivo `config.py`:

```python
# Provedor LLM
LLM_PROVIDER = 'lm_studio'        # 'ollama' ou 'lm_studio'
LLM_MODEL = 'bonsai-8b'
LLM_TEMPERATURE = 0.7
LLM_NUM_PREDICT = 300

# Assistente
ASSISTANT_NAME = "Chica"
ASSISTANT_GREETING = " Eu sou a Chica, sua assistente. Como posso ajudar?"

# Avatar (opcional)
AVATAR_ENABLE = True               # False desabilita o avatar

# Wake words
WAKE_WORDS = ["olá chica", "ei chica", "chica", ...]

# Voz TTS
TTS_VOICE = 'pf_dora'             # Kokoro (português)
TTS_SYSTEM = 'kokoro'             # 'kokoro' ou 'qwen3'
TTS_SAMPLE_RATE = 24000

# Sensibilidade do microfone
SPEECH_THRESHOLD = 0.005
SILENCE_DURATION = 1.5
INACTIVITY_TIMEOUT = 15.0

# Whisper (STT)
WHISPER_MODEL = 'base'             # 'tiny', 'base', 'small', 'medium', 'large'
```

🚀 EXECUTANDO O PROGRAMA
===================================================

1. Certifique-se que o provedor LLM está rodando:
   - Ollama: `ollama serve`
   - LM Studio: App aberto com API habilitada

2. Execute:
   ```bash
   python app.py
   ```

3. Na inicialização, escolha o sistema TTS:
   - 1 = Kokoro-TTS (padrão, português)
   - 2 = Qwen3-TTS (voz Serena)
   - 3 = Usar config.py

4. Comandos de voz:
   - "olá chica" — Acordar a assistente
   - Perguntas normais — Conversação com IA
   - "abra o navegador" — Executar comando local (confirma com "sim")
   - "calado" ou "silêncio" — Interromper a fala
   - Silêncio de 15 segundos — Volta a dormir

🗣️ COMANDOS LOCais
===================================================

A Chica pode executar comandos no seu sistema com confirmação por voz:

   "Chica, abra o navegador"   → "Confirma a abertura do navegador?" → "sim"
   "abra o Spotify"            → Abre o Spotify
   "abra o VS Code"            → Abre o Visual Studio Code
   "abra o terminal"           → Abre o terminal
   "abra o Discord"            → Abre o Discord
   "abra o Firefox"            → Abre o Firefox
   "abra a calculadora"        → Abre a calculadora

Comandos suportados em macOS, Linux e Windows.

🧠 MEMÓRIA PERSISTENTE
===================================================

A Chica lembra de informações importantes entre conversas:

- Salva automaticamente fatos relevantes (nome, preferências)
- Armazenamento em `chica_memory.json` (máx 10 memórias)
- Expira após 30 dias
- Injetada no contexto sem prejudicar desempenho
- Extração a cada 5 interações via LLM

🎯 DICAS DE USO
===================================================

1. Fale claramente e em um ambiente silencioso
2. Aguarde o áudio de resposta antes de falar de novo
3. Use "calado" para interromper a fala
4. O avatar ajuda a visualizar quando está falando
5. Para Raspberry Pi: use WHISPER_MODEL='tiny' e AVATAR_ENABLE=False

📦 RASPBERRY PI
===================================================

Recomendações para rodar no Raspberry Pi 4:

1. Use Whisper 'tiny' (mais leve): WHISPER_MODEL = 'tiny'
2. Desabilite o avatar: AVATAR_ENABLE = False
3. Use LM Studio com modelos pequenos (2B-4B parâmetros)
4. Ou use Ollama com modelos como qwen3:0.6b ou llama3.2:1b
5. Desabilite o thinking: THINKING_ENABLED = False
